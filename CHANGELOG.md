# Changelog

## [0.1.0](https://github.com/UI5/webcomponents-mcp-server/compare/v0.0.1...v0.1.0) (2026-03-05)

MCP server for UI5 Web Components development assistance.

### Features
- `get_component_api` - Fetch API docs for any UI5 Web Component
- `get_guidelines` - Integration guides for React, Angular, or native JavaScript
- `list_docs` - List available documentation
- `get_doc` - Fetch documentation content
